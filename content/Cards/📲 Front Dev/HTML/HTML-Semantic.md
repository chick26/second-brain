# 什么是语义元素？

**非语义**元素的例子：`<div>`和`<span>`- 没有说明它的内容。
**语义**元素示例： `<form>`、`<table>`和`<article>`- 明确定义其内容。

# HTML 中的语义元素

许多网站包含 HTML 代码，例如： <div id="nav"> <div class="header"> <div id="footer"> 来指示导航、页眉和页脚。

在 HTML 中，有一些语义元素可用于定义网页的不同部分： 
-  <文章>
-  <旁白>
-  <详情>
-  <图说明>
-  <图>
-  <页脚>
-  <标题>
-  <主>
-  <标记>
-  <导航>
-  <部分>
-  <摘要>
-  <时间>
